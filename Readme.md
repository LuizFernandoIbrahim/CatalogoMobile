Aplicativo mobile acadêmico desenvolvido com React Native e Expo Go, com foco em simular um e-commerce simples. O sistema possui autenticação com login, cadastro e biometria, armazenamento local de dados, catálogo de produtos com filtros, perfil editável com uso de câmera via Expo Camera, além de uma área administrativa para cadastro de produtos. Também inclui integração com mapa utilizando Expo Location, exibindo um endereço pré-definido.

Para iniciar o projeto:
- npm install
- npx expo start

Para realizar os testes:
Jest - npm test
Maestro - maestro test maestro/
