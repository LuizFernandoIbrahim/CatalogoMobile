Oficina de Marias — App Catálogo

Aplicativo mobile React Native (Expo) para catálogo da loja de artesanato Oficina de Marias.

Funcionalidades implementadas
- Tela de Login com autenticação biométrica (digital e Face ID)
- Detecção automática do tipo de biometria disponível
- Tela Home inicial com estrutura do catálogo
- Categorias
- Cards de produtos em destaque
- Identidade visual artesanal


Instalação

```bash
# 1. Instalar dependências
npm install

# 2. Iniciar o projeto
npx expo start

# Para iOS (Emulador)
npx expo start --ios

# Para Android (Emulador)
npx expo start --android
